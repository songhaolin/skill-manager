### 1. 📖 文档管理功能

## 🔧 技术细节

### 文档管理器 (DocumentManager)

**主要方法：**
```python
show_skill_info(skill_name)        # 显示文档选择菜单
get_management_doc(skill_name)     # 获取管理文档
get_source_readme(skill_name)       # 获取源码 README
show_comparison(skill_name)         # 对比显示
generate_management_doc(skill_name)  # 自动生成文档
verify_install_command(skill_name)   # 验证安装命令
```

**已验证的安装命令：**
```python
verified_commands = {
    "find-skills": "npx skills find",
    "brainstorming": "npx skills add obra/superpowers@brainstorming -g -y",
    "literary-story": "git clone ~/.claude/skills/literary-story",
    # ... 更多命令
}
```

### 更新检查器 (UpdateChecker)

**主要方法：**
```python
check_for_updates(force)           # 检查更新
should_check()                     # 判断是否应该检查
is_newer_version(latest, current)  # 比较版本号
get_update_notification()          # 获取更新通知
update_skill_manager()             # 一键更新
```

**配置选项：**
```python
{
  "update_notification": true,      # 是否显示通知
  "auto_check_interval": 7,         # 自动检查间隔（天）
  "last_check": "2026-03-02",       # 上次检查时间
  "last_version": "1.1.0"            # 上次版本
}
```
