## 📢 发布指南

### 步骤1：准备GitHub账户

1. 登录 https://github.com
2. 确保你的用户名（将YOUR_USERNAME替换为实际用户名）

### 步骤2：创建GitHub仓库

1. 访问 https://github.com/new
2. 设置：
   - Repository name: `skill-manager`
   - Description: `智能技能助手 - 说出需求，自动推荐安装技能`
   - Public: ✅ 选择公开
   - ⚠️ 不要勾选 "Add a README file"
3. 点击 "Create repository"

### 步骤3：修改配置

在 `skill-manager/` 目录下：

**Windows用户**：
```bash
# 编辑 publish.bat，修改第7行：
set USERNAME=你的GitHub用户名

# 或设置环境变量：
set GITHUB_USERNAME=你的GitHub用户名
```

**Linux/Mac用户**：
```bash
# 编辑 publish.sh，修改第10行：
USERNAME="${GITHUB_USERNAME:-你的GitHub用户名}"

# 或设置环境变量：
export GITHUB_USERNAME=你的GitHub用户名
```

### 步骤4：运行发布脚本

```bash
cd skill-manager

# Windows
publish.bat

# Linux/Mac
chmod +x publish.sh
./publish.sh
```

### 步骤5：创建Release

1. 访问你的GitHub仓库
2. 点击 "Releases"
3. 点击 "Create a new release"
4. 填写：
   - Tag: `v1.0.0`
   - Title: `🎉 skill-manager v1.0.0 - 智能技能助手`
   - Description: 复制 `RELEASE_NOTES.md` 的内容
5. 点击 "Publish release"
